Problem Set One

  • Give one example each of lexical and structural ambiguity in a language other than English, with enough explanation that a non-speaker of the language can understand the ambiguity.

Example (lexical)

kore-wanandesuka
DETCMDEM/NOUNVERBSFP
thisTOPwhat/naanCOPQ
‘What is this?’
‘Is this naan (bread)?’

In Japanese nan is ambiguous when spoken between an interrogative (何 "what") and Naan (ナン "Indian Bread"), and many other things. The ambiguity is normally resolved when written.
